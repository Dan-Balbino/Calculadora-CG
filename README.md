#Cálculo do Centro de Gravidade e Momento de Inércia de Figuras 2D
Este projeto foi desenvolvido para calcular o centro de gravidade e o momento de inércia de figuras geométricas bidimensionais (2D), aprimorando uma solução apresentada em um exercício de engenharia estrutural. O código oferece uma abordagem eficiente para determinar as propriedades físicas de formas 2D, considerando diferentes materiais e densidades.

Funcionalidades:
Cálculo do centro de gravidade de figuras compostas por formas geométricas simples em 2D (retângulos, triângulos, círculos, etc.).
Cálculo do momento de inércia de figuras 2D, importante para análise de resistência e comportamento estrutural.
Suporte para diferentes densidades e materiais, permitindo a análise de peças compostas.
Estrutura modular, facilitando a expansão para incluir novas formas geométricas.
Código otimizado para fácil compreensão e uso em projetos de engenharia.
Aplicações:
Este código é ideal para projetos de engenharia estrutural e design de peças mecânicas 2D, em situações onde é necessário calcular o centro de gravidade e o momento de inércia de estruturas planas.
